# Installing and Running

Follow the sections bellow to successfully install and run the project.

## Installing

To install, use `npm`:

    npm install torc_eval

## Running

To run, use: 

`npm run cy:run` or

`npm run cy:run-chrome`
`npm run cy:open` and select the test to be run
